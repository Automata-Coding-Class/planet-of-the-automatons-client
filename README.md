# Planet of the Automatons - CLI Client

This app accomplishes nothing on its own, really. It requires the game server to be running and reachable in order to fulfill its purpose. See https://github.com/Automata-Coding-Class/planet-of-the-automatons-complete for full details of the context for this app.

## Version History

*v0.0.1 - 22 Jan 2019* The initial release, used in the pilot session on 18 Jan 2019, plus some minor changes based on learnings from that session.
